Scratch

With Scratch, you can program your own interactive stories, games, and animations.

Scratch helps young people learn to think creatively, reason systemically, and work collaboratively - essential skills for life in the 21st century.

Scratch is a project of the Lifelong Kindergarten Group at the MIT Media Lab. It is provided free of charge.

For more information about Scratch, please visit the About Scratch page on the MIT site.

HTML

Hypertext Markup Language (HTML) is the language we structure our content in on the World Wide Web.

The first version of HTML was described by Sir Tim Berners-Lee, the inventor of the World Wide Web in 1989.

Since then, it has become the cornerstone technology used in the ever-expanding World Wide Web (including this very website).

For more information about HTML, please visit Introduction to HTML & CSS on the Code Club projects site.

Python

Python is a widely used, general purpose, text-based programming language.

It is the most complex of the languages we cover in Code Club, and is used by many professional programmers and companies.

For more information about Python, please visit About Python, and introductory notes are also available on the Code Club site.
